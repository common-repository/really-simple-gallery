== Description ==

A very simple to use gallery. You can upload your images and create a gallery page and a random image widget, which selects random images from your gallery.

== Installation ==

The gallery itself can be displayed by creating a new page and using the following code.

 <?php really_simple_gallery_display(); ?>

== Frequently Asked Questions ==

= Will more effects be added? =

* Yes, because the graphics library used has many effects already implemented and therefore, a simple update will add more effects.

= How to display the gallery on a page?  =

* You can simple create a new page and use the function really_simple_gallery_display() to display the gallery
